#mquery
===========

Expressive MongoDB query builder.

_work in progress_

## Installation

via npm:

    $ npm install mquery

Goals:

- flexible
- consistant API
- customizable
- browser compatibile
- compatibile with any driver (node-mongodb-native, mongo shell, etc)
