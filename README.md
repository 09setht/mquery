#mquery
===========

Expressive MongoDB query builder.

_work in progress_

## Installation

via npm:

    $ npm install mquery

Goals:

```
[x] flexible
[x] consistant API
[x] customizable
[ ] browser compatibile
[x] compatibile with node-mongodb-native
[ ] compatibile with mongo shell
```
